# What To Do When a Task is Completed

1. Run `npm run build` — ensure the project builds without errors
2. Run `npm run lint` — ensure no linting errors
3. Run `npm test` — ensure all tests pass
4. If build or lint fails, fix errors before proceeding
5. Commit with Conventional Commits format (`feat:`, `fix:`, etc.)
6. Use `git add .` then commit
7. Push to `main`
